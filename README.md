# Spatial-tumor-immune-continuum-in-DLBCL
Macrophage-rich dysfunctional inflammation along a spatial tumor-immune continuum in diffuse large B-cell lymphoma
DLBCL submission package - BCJ-ready v11

This package is a Blood Cancer Journal-oriented cleanup of the v10.9 enhanced package.

Key changes:
- Main manuscript abstract revised to a non-structured 145-word abstract.
- Main figures reduced to eight numbered figures for BCJ-style submission.
- Internal version language removed from the manuscript and supplementary materials.
- MCP-counter, F1_6 downsampling/block-level summary and spatially constrained LR scripts added to 06_scripts.
- MCP-counter p/FDR underflow values are reported as <2.2e-16 rather than 0.
- xCell-like analyses are explicitly described as marker-based proxy modules, not formal xCell package output.
- Supplementary Figure S59 description corrected.
